# repair-manager

> Приложение для управления клиентами и контролем потока работы.

## В проекте использованы следующие технологии:

### Front-end:
- React,react-router
- Redux,redux-thunk
- Material-ui
- axios

### Back-end:
- express
- mongoose
- passport
- bcrypt
- jwt
